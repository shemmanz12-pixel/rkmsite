RKM site (GitHub Pages)

Steps to publish:
1) Create a repo called rkm-site on GitHub.
2) Upload all files from this folder (index.html, style.css, and the other .html pages).
3) Go to Settings → Pages.
4) Under Build and deployment, set Source = Deploy from a branch.
5) Set Branch = main and /root. Click Save.
6) After 1–2 minutes, your site will be live at:
   https://YOURUSERNAME.github.io/rkm-site/

Use these in Google Ads sitelinks (replace YOURUSERNAME):
